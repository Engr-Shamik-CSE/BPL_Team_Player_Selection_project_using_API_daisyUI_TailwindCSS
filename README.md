# BPL Team Player Selection

A React project to select players for BPL teams using an API, styled with TailwindCSS + daisyUI.

## Features
- Fetch players from API
- Select & filter players
- Responsive UI with daisyUI cards and buttons

## Tech Stack
- React 18
- TailwindCSS
- daisyUI

## Run Locally
```bash
npm install
npm run dev
